### Epic Description

- **Story Key:** SCRUM-AAA
- **Title:** Apply Business Logic Filters for Expired and Inactive Discounts
- **Description:**

  - **Scenario:** Filtering expired discounts

    - **Given** the `expired` parameter is true
    - **When** a GET request is made

## API Documentation

### Endpoint
- **Path:** `/api/v1/discounts`
- **Method:** GET
- **Authentication:** Bearer JWT required

### Query Parameters
| Name         | Type    | Required | Description                                                      |
|--------------|---------|----------|------------------------------------------------------------------|
| productSku   | string  | No       | Product SKU to filter discounts                                  |
| market       | string  | No*      | Market code (required if productSku is present)                  |
| expired      | boolean | No       | Include expired discounts if true                                |
| ...          | ...     | ...      | Additional filtering parameters as per schema                    |

### Response Schema
```
[
  {
    "id": "string",
    "name": "string",
    "partners": ["string"],
    "product_models": [
      { "sku": "string", "name": "string" }
    ],
    "plan_benefits": [
      {
        "unit": "string",
        "value": "number",
        "plan": "string",
        "plan_id": "string",
        "adjustment": "number"
      }
    ],
    "end_date": "ISO8601 string",
    "inactive": "boolean",
    ...
  }
]
```

### Error Responses
| HTTP Code | Message                        | Scenario                                              |
|-----------|--------------------------------|-------------------------------------------------------|
| 400       | Market is required             | productSku present but market missing                 |
| 401       | Unauthorized                   | Missing or invalid JWT token                          |
| 200       | []                             | No discounts match filters                            |
| 422       | Validation failed              | Discount entity validation error                      |
